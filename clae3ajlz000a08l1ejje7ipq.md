## What is Web Development?
Before discussing the future scope of web development let's first understand what is web development.
> Web Development is a process of creating and maintaining web pages. These web pages combined we call a Website.

There are two major aspects of Web Development.
1. Frontend Development
2. Backend Development

## 1. Frontend Development
Whatever users see on the screen in the form of text, images, buttons, forms, tables, etc is the frontend part of the website.
This is a part of the website or webpage with which users can interact.
- HTML is used to structure the data on a webpage in various forms like text, image, video, audio, etc.
- CSS is used to design, style, or make the HTML structure more presentable and attractive.
- JavaScript is used to add events like click, hover, drag, etc, and makes web pages more interactive.

## 2. Backend Development
The backend is a part of the website that users can not see or can not directly interact with it.
Basically, it is used to store and organize the data.
- PHP, Java, Python, etc are a few of the languages used for backend development.

## Need for Web Development
The main purpose of web development is to make websites. The reasons for building websites could be thousands.
- It can be used to represent your business and your brand online.
- You can establish an online presence and build awareness about the products and services you offer.
- You can help people to understand how they can use those products and services to resolve their problems.

## Web Development as a career
There are thousands of web developers job available in industry for those who have a good understanding of skills.
If you want to be a web developer, make sure you learn all the skills required for that.
Below are a few you can go for related to web development
- Web App Developer
- UI Designer
- Frontend or Backend Developer
- UX Architect
- UX Designer
- Web Marketing Analyst

Apart from a job, you can also work as a freelancer or can start your own company.
Web Development can be an excellent, lucrative career, if you are passionate about developing websites, web apps, etc.
You need to frequently upgrade your skills as it is a continuously evolving field.
